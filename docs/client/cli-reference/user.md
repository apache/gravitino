## User commands 

### List all users

```bash
gcli user list
```

### Create a user

```bash
gcli user create --user new_user
```

### Show a user's details

```bash
gcli user details --user new_user
```

### Show a user's audit information

```bash
gcli user details --user new_user --audit
```

#### Delete a user

```bash
gcli user delete --user new_user
```
