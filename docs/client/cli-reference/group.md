## Group commands 

### List all groups

```bash
gcli group list
```

### Create a group

```bash
gcli group create --group new_group
```

### Display a group's details

```bash
gcli group details --group new_group
```

### Show a groups's audit information

```bash
gcli group details --group new_group --audit
```

### Delete a group

```bash
gcli group delete --group new_group
```
